# MyClient2
Flash ActionScript3 2D游戏引擎，基于四叉树数据结构实现，支持格斗类ACT游戏和角色扮演类ARGP游戏，诞生于2010年，作者王明凡

目前的版本的adobe flash和adobe air是无法运行该项目，需要2012年以前的旧版本才能运行该项目
