Package com.myclient2.interfacesInterfaces
 


Interfaces
 InterfaceDescription
 IEngine IEngine接口定义了用于渲染地图显示对象(也可以称作贴图)，以及操作地图显示对象
 ILoader ILoader接口定义了加载外部文件的方法，可以采用二进制或URL方式加载jpg,png,swf或其他文件
 IMap IMap接口定义了用于创建一个地图对象，将显示对象添加到地图对象中显示
 IOpenMap IOpenMap接口定义了采用二进制方式打开一个地图文件(.mcmap),并返回地图的材质对象集合和地图信息
 IQuadtree IQuadtree接口定义了用于操作(创建，查询，删除)四叉树数据结构
 IQuadtreeNode IQuadtreeNode接口定义四叉树每个节点对象，描述一个四叉树节点对象信息，位置，范围等
 IRectangle IRectangle接口按其位置（由它左上角的点 (x, y) 确定）以及宽度和高度定义的区域
 ISearch ISearch定义了寻路,用于对直角路点数组进行寻路
 ISprite ISprite接口统一定义了IMap中显示对象
 ITiles ITiles接口定义显示到IMap中的平铺地砖对象，平铺整个IMap
